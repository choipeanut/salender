# Product Spec (Draft)

## 목표

여러 브랜드의 세일 이벤트를 수집해 월/주/리스트 기반으로 통합 조회할 수 있는 모바일 경험을 제공한다.

## 핵심 사용자 가치

- 브랜드별 세일 일정 확인
- 관심 브랜드 구독
- 세일 상태 변화 알림 수신

## MVP 범위 요약

- Expo 모바일 앱
- Supabase 백엔드
- 브랜드 4개 등록
- real connector 1개 + mock/manual 데이터
- 캘린더 보기, 브랜드 선택/구독, 알림 기본 기능

## 비범위

- 결제/구매 플로우
- 로그인 우회/비인가 접근/anti-bot 회피
